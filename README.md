# Juntos a Jugar
## Ingenieria de SW Avanzado

Front-End React App de Juntos a Jugar.

### Requisitos

- npm.
- API juntos a jugar iniciada en el `http://localhost:3000`

### Instrucciones

- Clonar este repositorio.
- `npm start`

### Deploy
- Clonar este repositorio
`cd juntosajugar_front`
`npm install`
`npm run build`
`npm install`
`npm install -g serve`
`serve -s build`
